# Algorithms and Data Structures 2 (AED2) 📊

Academic project developed for the **AED2** course at the University of Minho.

### 📋 Description
This project focuses on managing orders (expedited and pending) and maintaining a history log, applying advanced data structure concepts in C.

### 🛠️ Technologies
* **Language:** C

### 📂 Structure
* `src/`: Source code (`main.c`) and data files (`.txt`).
