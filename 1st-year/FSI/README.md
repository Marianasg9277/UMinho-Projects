# Information Systems Fundamentals (FSI) 📊

Academic project developed for the **FSI** course at the University of Minho, focusing on the analysis and modeling of information systems.

### 📋 Description
This project was developed in progressive stages, covering the semantic and semiotic analysis of information systems, as well as structural modeling.

### 📂 Stages
* **FSI_Análise Semântica**: Semantic analysis phase.
* **FSI_Análise Semiótica**: Semiotic analysis phase.
* **FSI_Organigrama**: Organizational structure modeling.
* **FSI_Representação de Sistemas de Informação**: Final representation of the system.
