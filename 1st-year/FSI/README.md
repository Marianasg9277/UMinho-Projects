# Café Desgraça - Information Systems Modeling ☕

Academic project developed for the **Fundamentos dos Sistemas de Informação (FSI)** (Information Systems Fundamentals) course at the University of Minho (2024/2025).

### 📋 Description
A comprehensive study and detailed modeling of the information system and business processes for *Café Desgraça*, a local establishment and strategic meeting point for university students. The project breaks down the business ecosystem into organizational, semantic, and behavioral layers.

### 📂 Project Documents & Analysis
The detailed reports for each phase of the project are documented in the following files:

* 📄 **[`FSI_Análise Semântica.pdf`](./FSI_Análise%20Semântica.pdf) & [`FSI_Análise Semiótica.pdf`](./FSI_Análise%20Semiótica.pdf):** 
  Ontological mapping of the business ecosystem and detailed semiotic analysis of informational objects (e.g., digital invoices) covering physical, syntactic, semantic, and pragmatic dimensions.

* 📄 **[`FSI_Organizagrama.pdf`](./FSI_Organizagrama.pdf):** 
  Organizational structuring and process categorization using **Porter's Value Chain** and the standard **APQC Process Classification Framework**. Includes Entity-Relationship Diagrams (ERD) and a **Balanced Scorecard (BSC)**.

* 📄 **[`FSI_Representação de Sistemas de Informação.pdf`](./FSI_Representação%20de%20Sistemas%20de%20Informação.pdf):** 
  Behavioral modeling of the business logic mapping interactions between clients, suppliers, and internal systems through multiple UML diagrams (Use Case, Activity, Sequence, and State Transition).

### 🛠️ Concepts & Tools Applied
* **Business Frameworks:** Porter's Value Chain, APQC Framework, Balanced Scorecard (BSC).
* **System Modeling (UML):** Use Case Diagrams, Activity Diagrams, Sequence Diagrams, and State Transition Diagrams.
* **Data Modeling:** Entity-Relationship Diagram (ERD).
* **Theoretical Analysis:** Semiotics, Ontologies, and Informational Object Analysis.
